# HBDb
 All things films. Search, Discover, Rate, Watch Later

Dependencies:
- npm
- allow CORS browser plugin (until web service is developed)
  - Tested with Firefox: https://addons.mozilla.org/en-GB/firefox/addon/access-control-allow-origin/

Instructions:
- Clone repo
- cd into repo
- npm install
- npm run build
- npm run dev
